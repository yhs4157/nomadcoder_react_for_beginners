*** npm vs npx 
 - npm은 node Package manager
 - npx은 node package executer

 - npm은 버전의 유지관리한다 그렇기에 버전이 여러가지 갈래로 벌어지는데 반해
 npx는 작동만 중시하기에 버전은 관리 않고 가장 최신의 버전만 들고 온다. 

 git에 다 올리지 않고 바뀌는 부분만 올리면서 상황을 계속 주시하자.
